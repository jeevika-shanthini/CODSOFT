# Python Mini Projects 🐍

A collection of beginner-friendly Python console applications built to practice core programming concepts like loops, functions, dictionaries, lists, and user input handling.

## 📂 Projects

### 1. 🎮 Rock Paper Scissors Game
A simple command-line game where the user plays Rock-Paper-Scissors against the computer.
- Random computer choice using the `random` module
- Win/lose/draw logic
- Play-again loop until the user chooses to quit

**File:** `rock-paper-scissors_game.py`

---

### 2. 🔐 Password Generator
Generates a random, secure password based on user-specified counts of letters, numbers, and symbols.
- Customizable number of letters, numbers, and symbols
- Randomly shuffles characters for added security
- Combines selections into a final password string

**File:** `Password_Generator.py`

---

### 3. ✅ To-Do List
A menu-driven task manager for adding, viewing, removing, and completing tasks.
- Add new tasks
- View all tasks with status (Pending/Done)
- Remove a task by number
- Mark a task as completed
- Runs in a continuous menu loop until exit

**File:** `To-Do-List.py`

---

### 4. 📇 Contact Book
A menu-driven contact management system using a dictionary to store contact details.
- Create, view, update, and delete contacts
- Search contacts by (partial) name
- Count total number of saved contacts

**File:** `Contact_book.py`

---

### 5. 🧮 Simple Calculator
A basic calculator supporting the four fundamental arithmetic operations.
- Addition, Subtraction, Multiplication, Division
- Handles division-by-zero errors
- Menu-driven interface

**File:** `Calculator.py`

---

## 🚀 How to Run

Each project is a standalone script. Run any of them with:

```bash
python <filename>.py
```

Example:
```bash
python Calculator.py
```

## 🛠️ Requirements

- Python 3.x
- No external libraries required (uses only built-in modules: `random`, `math`)

## 📌 Purpose

These projects were built as hands-on practice for Python fundamentals — control flow, functions, data structures (lists/dictionaries), and basic input validation — through small, self-contained applications.
